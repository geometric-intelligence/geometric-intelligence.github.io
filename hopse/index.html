Abstract
While Graph Neural Networks (GNNs) have proven highly effective at modeling relational data, pairwise connections cannot fully capture multi-way relationships naturally present in complex real-world systems. In response to this, Topological Deep Learning (TDL) leverages more general combinatorial representations —such as simplicial or cellular complexes— to accommodate higher-order interactions. Existing TDL methods often extend GNNs through Higher-Order Message Passing (HOMP), but face critical scalability challenges due to (i) a combinatorial explosion of message-passing routes, and (ii) significant complexity overhead from the propagation mechanism. To overcome these limitations, we propose HOPSE (Higher-Order Positional and Structural Encoder), a framework free of message passing layers that uses Hasse graph decompositions to derive efficient and expressive encodings over arbitrary higher-order domains. Notably, HOPSE scales linearly with dataset size while preserving expressive power and permutation equivariance. Experiments on molecular, expressivity and topological benchmarks show that HOPSE matches or surpasses state-of-the-art performance while achieving up to 7× speedups over HOMP-based models, opening a new path for scalable TDL.
Key Advantages

Why HOPSE?

Existing Topological Neural Networks suffer from a combinatorial explosion of message-passing routes — for a complex of dimension R, the number of routes scales as (R+1)!·(R+1)R. HOPSE sidesteps this entirely.

No Message Passing

All structure-dependent computations are offloaded to a one-time preprocessing step via Hasse graph decompositions. Training uses only lightweight MLPs — no iterative message propagation.

Linear Scaling

Complexity scales as O(LK Σ Nr Dr) — linear in the number of cells. This enables up to 7× speedups over HOMP-based models like GCCN and SCCNN.

Theoretical Guarantees

Provably more expressive than the CCWL isomorphism test (Theorem 4.1) while preserving cell permutation equivariance (Theorem 4.2) — without message passing during training.

Background

Key Concepts

Combinatorial Complexes

A combinatorial complex T = (V, C, rk) consists of cells organized by rank: nodes (0-cells), edges (1-cells), faces (2-cells), and higher. Neighborhood functions — adjacencies At,s and incidences Is→t — define how cells of different ranks interact.

Hasse Graph Decomposition

Each neighborhood function N induces a strictly augmented Hasse graph GN. HOPSE treats each Hasse graph as an independent component, enabling efficient application of established graph PSE techniques to higher-order domains.

Two Realizations

HOPSE-M

Manual Encodings

Hand-crafted graph-theoretical positional and structural encodings applied to each Hasse graph:

  • Laplacian Eigenvectors (LapPE)
  • Random Walk Structural Encodings (RWSE)
  • Heat Kernel Signatures (HKdiagSE)
  • Electrostatic Potential Encodings (ElstaticPE)
HOPSE-G

Pretrained GPSE

Leverages the pretrained GPSE network to encode structural and positional information in a single pass:

  • Single learned encoder g1 = GPSE(·)
  • Avoids costly eigendecomposition
  • Especially scalable for large complexes
  • Uses random noise initialization
