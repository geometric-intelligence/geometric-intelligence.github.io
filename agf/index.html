Overview

Alternating Gradient Flows (AGF) is a comprehensive research program that develops a theory of feature learning in neural networks through the lens of dynamical systems and group theory. This work addresses fundamental questions about how neural networks learn hierarchical representations and compositional structures.

The first paper, Alternating Gradient Flows: A Theory of Feature Learning in Two-layer Neural Networks, establishes the foundational framework. It introduces AGF dynamics as a way to understand feature learning in two-layer neural networks through the alternating optimization of features and output weights. This framework reveals how networks progressively learn hierarchical features through a sequence of phase transitions, providing both theoretical guarantees and practical insights.

Building on this foundation, the second paper, Sequential Group Composition: A Window into the Mechanics of Deep Learning, extends AGF theory to group-theoretic learning tasks. It studies how neural networks learn to compose group operations sequentially, discovering that networks learn one irreducible representation at a time in a characteristic "staircase" pattern. This work provides a bridge between the abstract theory of AGF and concrete mechanistic interpretations of how networks solve compositional tasks.

Together, these papers establish AGF as a powerful lens for understanding feature learning, representation theory, and compositional reasoning in deep learning systems.

Key Results & Figures

1. The AGF Framework for Feature Learning

The core insight of AGF is that feature learning in two-layer networks can be understood through alternating optimization: first optimizing features while keeping output weights fixed, then optimizing output weights while keeping features fixed. This decomposition reveals the underlying dynamics of how networks progressively learn better representations.

Figure 1 (Paper 1): Overview of Alternating Gradient Flows. The learning process alternates between feature optimization (adjusting the first layer weights) and output weight optimization (adjusting the second layer). This decomposition allows us to track how features evolve over time and understand the phase transitions in the learning dynamics.

2. Phase Transitions and the Role of Initialization

AGF theory provides rigorous convergence guarantees showing when and how networks will learn meaningful features. A key finding is the critical role of initialization scale β in determining the learning dynamics. Networks trained with different β values exhibit distinct learning trajectories, with smaller β leading to slower, more sequential feature discovery.

Figure 2 (Paper 1): Effect of initialization scale β on learning dynamics. Different initialization scales lead to different learning trajectories, with the loss exhibiting distinct phase transitions. The AGF framework accurately predicts these transitions across the full range of β values.

3. Feature Learning in Transformers

AGF theory extends beyond simple feedforward networks. We apply the framework to transformers learning modular arithmetic, showing that the same alternating dynamics govern feature learning in attention mechanisms. The theory predicts the structure of learned features, which can be characterized through the singular value decomposition of attention weight matrices.

Transformer Singular Values: Evolution of singular values showing how attention heads learn structured features over time

4. Sequential Group Composition: A New Lens

Extending AGF to group-theoretic tasks, we study how networks learn to compose group operations. The sequential group composition task requires networks to learn multiple group representations and compose them in sequence—a fundamental capability for compositional reasoning. This task provides a controlled setting to study how networks discover and utilize mathematical structure.

Figure 3 (Paper 2): Overview of the sequential group composition framework. Networks must learn to compose a sequence of group operations g₁ ∘ g₂ ∘ ... ∘ gₙ, where each operation comes from a finite group. The figure illustrates how different architectural choices (two-layer, recurrent, multilayer) affect the learning dynamics and efficiency.

5. The Staircase Pattern: Learning Irreps Sequentially

Our most striking finding is that networks learn group representations through a characteristic "staircase" loss pattern. Each step in the staircase corresponds to the network discovering and learning one irreducible representation (irrep) of the group. This provides direct mechanistic insight into how networks decompose compositional structure. The staircase pattern is universal, appearing consistently across different group types (cyclic, dihedral, symmetric) and network architectures.

Figure 4 (Paper 2): The characteristic "staircase" loss pattern in group composition learning. Each plateau-to-drop transition represents the network discovering and learning a new irreducible representation. The figure shows results across multiple group types, demonstrating the universality of this learning pattern. The staircase structure reveals that networks learn group representations one irrep at a time, in an order determined by the Fourier statistics of the data encoding.

6. Bridging Theory and Practice

The group composition results in Paper 2 directly validate and extend the AGF framework from Paper 1. The staircase pattern observed in group learning is a specific manifestation of the sequential feature discovery predicted by AGF theory. Both papers demonstrate that neural networks learn hierarchical representations through discrete phase transitions, whether those representations are data-driven features or group-theoretic irreducible representations. This unified perspective provides a deeper understanding of the fundamental mechanisms underlying deep learning.

Code & Reproducibility

All results in both papers are fully reproducible. We provide two comprehensive code repositories with Jupyter notebooks, experiment scripts, and detailed documentation.

AGF Code

Theory and experiments on AGF dynamics in two-layer networks. Includes implementations for linear networks, transformers on modular arithmetic, and various feature learning tasks.

View Repository

Group-AGF Code

Experiments on sequential group composition. Includes implementations for multiple group types (cyclic, dihedral, symmetric) and analysis tools for studying the staircase learning pattern.

View Repository

Getting Started

Both repositories include Jupyter notebooks that walk through the main experiments and provide interactive visualizations. Clone either repository and follow the README instructions to reproduce our results or extend the framework to new tasks.
